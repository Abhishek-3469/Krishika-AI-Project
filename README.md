# Krishika - Smart Farming Assistant

Krishika is a comprehensive digital platform designed to empower Indian farmers with AI-driven insights, crop management tools, and a supportive community.

## Features

- **AI Chatbot (Krishika AI):** Get instant answers to farming queries in English, Hindi, Telugu, and Odia.
- **Crop Yield Predictor:** Estimate potential harvest based on input parameters.
- **Soil Report Analyzer:** Upload soil health reports for detailed analysis and recommendations.
- **Community:** Connect with other farmers, share experiences, and get advice.
- **Dashboard:** Track productivity, expenses, and farm activities.

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **UI Framework:** Tailwind CSS, shadcn/ui
- **AI Integration:** Google Gemini API

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/mukulgit649/harit-digi.git
    cd harit-digi
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Set up Environment Variables:
    Create a `.env` file in the root directory and add your Gemini API key:
    ```env
    VITE_GEMINI_API_KEY=your_api_key_here
    ```

4.  Run the development server:
    ```bash
    npm run dev
    ```
